# SynthEdge — Technical Explanation

## What It Does
SynthEdge is a real-time trading terminal that connects **Synth's probabilistic price forecasts** to **Hyperliquid's on-chain equity perpetual execution**. Users bring their own Synth API key and Hyperliquid wallet, then get AI-powered probability distributions, volatility analytics, and risk management — with one-click trading directly on Hyperliquid's equity perps (NVDA, TSLA, AAPL, GOOGL, SPY) and crypto perps (BTC, ETH, SOL).

## How It Uses the Synth API

**Primary endpoint:** `/insights/prediction-percentiles` — polled every 60 seconds per asset, cached in Redis. From this single endpoint, we locally derive: implied volatility (5th–95th percentile spread), directional probability (median vs current price), distribution skew and kurtosis, liquidation risk estimates, and probability cone visualizations.

**Dedicated insight endpoints** (called only when user navigates to specific views): `/insights/volatility` for the volatility deep-dive, `/insights/option-pricing` for the options view, `/insights/liquidation` for the Liquidation Guardian. This architecture minimizes API credit usage while maximizing the intelligence extracted.

**Assets covered:** All 9 Synth-supported assets across both 1-hour and 24-hour horizons — BTC, ETH, SOL, XAU, SPYX, NVDAX, TSLAX, AAPLX, GOOGLX.

## Probabilistic Modeling (30% of criteria)

Every feature is built on Synth's probability distributions, not point predictions:
- **VaR/CVaR** computed from Synth's distribution tails (not historical returns)
- **Kelly Criterion** position sizing using Synth-derived win/loss probabilities
- **Regime detection** from distribution shape: skew, kurtosis proxy, volatility width
- **Probability cones** render the full 9-percentile forecast across all timesteps
- **Tail risk monitoring** tracks extreme percentiles (0.5th, 5th) for downside exposure
- **Funding rate arbitrage** signals generated by comparing Synth's directional forecast against Hyperliquid's funding rates

## Technical Implementation (30% of criteria)

- **Frontend:** Next.js 15 (App Router) + TypeScript + Tailwind + TradingView Lightweight Charts + D3.js
- **Backend:** Python FastAPI (async) + Celery for periodic polling + Redis for caching/pub-sub
- **Databases:** PostgreSQL (trade journal, positions), MongoDB (forecast snapshots, time-series), Redis (live cache)
- **Performance:** Rust (PyO3) for Monte Carlo VaR and batch distribution math
- **Real-time:** WebSocket fan-out from Redis Pub/Sub to connected clients
- **Execution:** Hyperliquid Python SDK for reading positions; client-side TypeScript signing for trade execution (private keys never leave the browser)

## Practical Market Relevance (25% of criteria)

SynthEdge solves real problems for active traders:
1. **See the edge:** Probability cones, directional scanner, and volatility heatmaps surface Synth's intelligence visually
2. **Size correctly:** Kelly Criterion computes mathematically optimal position sizes from Synth's distributions
3. **Execute instantly:** One-click trading on Hyperliquid's equity perps, 24/7, with up to 20x leverage
4. **Manage risk:** Liquidation Guardian alerts, VaR monitoring, dynamic stop-losses from distribution percentiles
5. **Learn and improve:** Trade journal enriches every trade with Synth's forecast context for post-trade analysis

## Innovation (15% of criteria)

SynthEdge is the **first product bridging Synth's AI forecasts with Hyperliquid's on-chain execution**. The BYOK (Bring Your Own Key) model means we popularize both ecosystems — users need a Synth API key for intelligence and a Hyperliquid wallet for execution. Both platforms share near-perfect asset coverage and a crypto-native audience. The funding rate arbitrage scanner — comparing on-chain funding sentiment against Synth's probabilistic forecasts — is a novel signal type that only exists at this intersection.
