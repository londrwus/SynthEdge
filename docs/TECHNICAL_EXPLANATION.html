SynthEdge — Technical Explanation

Synth Hackathon 2026 · Best Equities Application · Live: synthedge.xyz · GitHub: londrwus/SynthEdge

What It Does

SynthEdge is a probabilistic trading terminal connecting Synth's AI-powered price forecasts with Hyperliquid's on-chain equity perpetual execution. Users bring their own Synth API key and Hyperliquid wallet, then see probability distributions, volatility analytics, regime detection, and risk management — with direct trade execution on Hyperliquid's perps and HIP-3 equity markets across 9 assets: BTC, ETH, SOL, XAU, SPY, NVDA, TSLA, AAPL, GOOGL.

How It Uses the Synth API

Primary endpoint: /insights/prediction-percentiles — polled every 5 minutes, cached in Redis (10-min TTL). Each response: 289 timesteps × 9 percentile levels (0.5th–99.5th). From this single endpoint we locally derive everything: implied volatility (P5–P95 spread annualized), directional probability, distribution skew/kurtosis, regime classification, tail risk metrics, and probability cone visualizations. Dedicated endpoints (/insights/option-pricing, /insights/liquidation) called on-demand only. Total: ~144 API credits/hour — 85% less than naive polling.

Probabilistic Modeling (30%)

Every feature uses probability distributions, not point predictions:

  • VaR/CVaR from distribution tails (forward-looking)
  • Kelly Criterion sizing from Synth-derived win/loss probabilities
  • Regime detection from distribution shape → 4 regimes with playbooks
  • Probability cones — 9 percentiles × 289 timesteps, interactive D3.js
  • Earnings vol — Synth forecast vs historical equity earnings moves
  • Smart orders — SL/TP from P80/P20 percentiles
  • Cross-asset vol — crypto vs equity ratio (~6x) from same model
  • Signals engine — direction, confluence, tail risk, vol spike alerts

Technical Implementation (30%)

  • Backend: Python 3.12 + FastAPI (async), 25 REST endpoints, Redis 7 caching with pipeline writes, GZip, connection pooling. 53 pytest tests.
  • Frontend: Next.js 15 + TypeScript, 12 routes, TradingView Lightweight Charts v5, D3.js probability cone, Zustand, TanStack Query v5.
  • Execution: Hyperliquid Python SDK — crypto perps (BTC/ETH/SOL) + HIP-3 spot (TSLA/AAPL/GOOGL/SPY). MetaMask wallet connect.
  • Infrastructure: Docker Compose (5 containers), Nginx, Let's Encrypt SSL, Ubuntu 24.04.

Practical Market Relevance (25%)

Addresses all 5 hackathon equity examples: (1) Earnings volatility dashboard comparing forecasts vs historical moves, (2) Cross-asset correlation tracking equities vs crypto vol, (3) Equity options strike selector from Synth's price ranges, (4) Portfolio risk monitor using Synth's vol forecasts, (5) Regime detection flagging vol shifts. Plus: directional scanner, Kelly sizer, signals & alerts, and direct Hyperliquid execution with Synth-derived SL/TP levels.

Innovation (15%)

First product bridging Synth's AI forecasts with Hyperliquid's on-chain execution. Key innovations: Synth-powered smart orders (SL/TP from percentiles, not arbitrary %), cross-asset vol ratio from the same AI model, regime-based trading playbooks from distribution shape, signals engine generating alerts from probabilistic data, and BYOK model driving adoption of both ecosystems.
